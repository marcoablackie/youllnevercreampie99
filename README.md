# zenbenyama is aura

NBA 2K26 custom jumpshot finder — pick your height and 3PT rating, get a full jumper recipe with the biggest green window.

## Live site

After enabling GitHub Pages on this repo (Settings → Pages → Deploy from `main` branch), the app will be at:

https://marcoablackie.github.io/youllnevercreampie99/

## Run locally

Open `index.html` in a browser, or serve the folder with any static file server.

## Data

Unlock requirements scraped from GameRant and NBA2KW. Custom build recipes use researched 2K26 meta guides and NBA2KLab public data.
